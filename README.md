## 📂 레포지토리 구조

각 섹션(Section)은 주제별 폴더로 구성되어 있으며, 각 폴더 안에는 문제별 패키지가 존재합니다.

```
.
├── sec01_string/
│   └── q01_find_character/
│       ├── MySolution.java  // 내 풀이 코드
│       ├── Answer.java      // 강사님 풀이 코드
│       ├── Review.java      // 다시 풀이
│       └── README.md        // 문제 분석 및 학습 내용 정리
├── sec02_array/
│   └── ...
└── ...
```
