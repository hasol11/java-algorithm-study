# 1. 문자 찾기

### 1) 로직 흐름

1.  입력 통일: 검색의 정확성을 위해 입력받은 문자열(String)과 찾을 문자(char)를 `toUpperCase()`를 이용해 모두 대문자로 변환한다.
2.  문자열 순회: `toCharArray()`로 문자열을 문자 배열로 바꾼 뒤, 향상된 for문을 사용해 배열의 각 문자를 처음부터 끝까지 순회한다.
3.  조건 검사 및 카운트: 순회 중인 문자가 찾으려는 문자와 일치하는지 검사한다. 일치하면, 미리 선언해 둔 카운트 변수(`answer`)의 값을 1 증가시킨다.
4.  결과 반환: 모든 문자를 순회한 뒤, 최종적으로 누적된 카운트 값을 반환한다.

---

### 2) 핵심 개념

- 향상된 for문 (Enhanced for loop): `for (char x : str.toCharArray())` 형태로 문자열의 각 문자를 간단하게 순회할 수 있다.
- 대소문자 변환: `toUpperCase()`나 `toLowerCase()`를 사용하면, 문제의 요구사항인 '대소문자를 구분하지 않는' 조건을 쉽게 만족시킬 수 있다.
