# 4. 단어 뒤집기

-> N개의 단어가 주어지면 각 단어를 뒤집어서 출력하는 프로그램을 작성

### 1) 로직 흐름

내 로직

1. n을 입력 받고, 입력받은 n 크기의 배열을 생성한다.
2. n번 입력을 받는데, 입력받을 때 저장하는 것을 for 문을 사용하여 거꾸로 Stringbuilder을 만들어서 저장한다.
3. 저장된 배열을 출력한다.

### 2) 핵심 개념

StringBuilder: String과 달리 내용을 변경할 수 있는(mutable) 문자열 클래스다. .reverse(), .append() 등 기능이 있고, 반복문 안에서 문자열을 합치거나 바꿀 때 사용하면 성능상 이점이 크다.

.reverse(): StringBuilder 객체가 가진 문자열의 순서를 뒤집어주는 매우 편리한 메서드다.

ArrayList<String>: 크기가 정해진 배열(String[])과 달리, 필요에 따라 크기가 자동으로 늘어나는 동적 배열이다. 데이터를 유연하게 추가(add)하거나 삭제할 수 있다.

향상된 for문 (Enhanced for-loop): for(String x : str)와 같은 형태로, 배열이나 리스트의 모든 요소를 처음부터 끝까지 간단하게 순회할 때 사용한다. 인덱스를 신경 쓸 필요가 없다.

추가로 `toCharArray()`를 사용해 문자열을 배열로 만든 후 lt, rt를 사용하여 직접 교환해 뒤집는 방법도 있다.
