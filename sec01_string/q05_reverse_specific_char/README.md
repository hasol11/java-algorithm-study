# 5. 특정 문자 뒤집기

-> 영어 알파벳과 특수문자로 구성된 문자열이 주어지면 특수문자는 그대로 두고 알파벳만 뒤집는 프로그램 작성

### 1) 로직 흐름

1. 초기화: 문자열을 직접 수정할 수 없으므로 toCharArray()를 사용해 문자 배열(char[])로 변환한다.
2. 포인터 설정:
   lt (Left Pointer): 0번 인덱스 (왼쪽 끝)
   rt (Right Pointer): length - 1번 인덱스 (오른쪽 끝)
3. 반복 탐색 (while(lt < rt)):
   왼쪽 확인: s[lt]가 특수문자면? → 뒤집을 필요 없으므로 lt++ (무시하고 전진).
   오른쪽 확인: s[rt]가 특수문자면? → 뒤집을 필요 없으므로 rt-- (무시하고 후진).
   둘 다 알파벳이면: 두 문자의 위치를 **교환(Swap)**하고, lt++, rt--로 안쪽으로 좁혀 들어간다.
4. 결과 반환: 수정된 문자 배열을 String.valueOf()로 다시 문자열로 변환하여 출력한다.

### 2) 핵심 개념

1. lt, rt 투 포인터 개념 사용
2. Character.isAlphabetic(char c) : 해당 문자가 알파벳인지 판별해주는 메서드
3. 문자열 <-> 문자 배열 교환
   `str.toCharArray()` : String을 수정 가능한 char[] 배열로 변환
   `String.valueOf(char[] s)` : char[] 배열을 다시 String으로 변환
