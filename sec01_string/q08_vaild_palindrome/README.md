# 8. 유효한 팰린드롬

-> 알파벳 이외의 문자들은 무시한 채로 유효한 팰린드롬인지 확인하는 프로그램 작성

### 1) 로직 흐름

1. 전처리

- `toUpperCase()` : 대소문자 구분을 없애기 위해 문자열을 모두 대문자로 변환한다.
- `replaceAll("[^A-Z]", "")` : 정규표현식을 사용해 알파벳 대문자가 아닌(^) 모든 문자를 빈 문자열로 치환하여 제거한다.

2. 문자열 뒤집기 및 비교 : `StringBuilder`의 `reverse()`를 사용해 정제된 문자열을 뒤집고, `equals()`로 같은지 확인한다.

---

### 2) 핵심 개념

- replaceAll(String regex, String replacement)
  단순한 문자열 치환(replace)과 달리, **정규표현식(Regex)**을 패턴으로 사용하여 문자열을 변환한다.
- 정규표현식 [^A-Z] 해석
  A~Z(대문자)가 아닌 모든 글자
