# 9. 숫자만 추출

-> 문자열에서 숫자만 추출하여 자연수를 만드는 프로그램 작성

### 1) 로직 흐름

1. 방법 1 : 정규표현식 & parseInt

- replaceAll("[^0-9]", "")을 사용하여 숫자가 아닌 문자를 전부 제거한다.
- Integer.parseInt()를 사용하여 문자열을 정수로 변환한다.

2. 방법 2: 아스키코드 & 수학적 계산

- 아스키코드 값 48('0') ~ 57('9') 사이인지 확인하여 숫자만 추출한다.
- 누적 공식 : answer = answer \* 10 + (x - 48)

---

### 2) 핵심 개념

1. 아스키코드와 숫자 변환 (x - 48)

- 컴퓨터에서 문자 '0'은 실제 숫자 0이 아니라 아스키코드 값 48이다. 따라서 문자(char)를 실제 정수(int) 값으로 바꾸려면 '0'(48)을 빼줘야 한다.

2. 자릿수 올림 공식 (answer \* 10)
