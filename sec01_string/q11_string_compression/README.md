# 11. 문자열 압축

-> 알파벳 대문자로 이루어진 문자열을 입력받아 같은 문자가 연속으로 반복되는 경우 반복되는 문자 바로 오른쪽에 반복 횟수를 표기하는 프로그램 작성

### 1) 로직 흐름

문자열을 0부터 length-1까지 순회한다.

- if (str.charAt(i) == str.charAt(i+1)): 현재 문자와 다음 문자가 같다면, 연속된 문자이므로 카운트(cnt)를 증가시킨다.
- else: 현재 문자와 다음 문자가 다르다면, 연속이 끝난 시점이다. 결과 문자열에 현재 문자(charAt(i))를 누적한다.만약 cnt > 1이라면, 카운트 숫자도 문자로 변환하여

---

### 2) 핵심 개념

1. 인덱스 에러 방지용 패딩

- 배열이나 문자열 탐색 시 i+1을 참조해야 할 때, 의미 없는 값(공백 등)을 끝에 추가하여 마지막 인덱스에서 범위 초과 에러를 막는다.

2. `String.valueOf()`

- 정수형을 문자열에 붙일 때 사용한다.
