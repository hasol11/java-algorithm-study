# 12. 암호

-> '\*'와 '#'으로 이루어진 암호를 각각 0,1로 변환하고, 7자리씩 끊어서 2진수 문자열을 만든다. 10진수로 변환 후 아스키코드 문자로 변환하여 원래 단어를 완성하는 프로그램 작성

### 1) 로직 흐름

1. 문자열 자르기 substring

- 입력받은 암호 문자열을 7글자씩 끊어서 처리한다.

2. 이진수 변환 replace

3. 십진수 및 문자 변환

- `Integer.parseInt(binaryStr, 2)를 사용하여 2진수 문자열을 10진수 정수로 변환한다.
- 변환된 정수를 (char)로 강제 형변환하여 해당하는 알파벳을 얻는다.

---

### 2) 핵심 개념

1. 2진수 → 10진수 변환
2. 메서드 체이닝
   replace 메서드를 연속으로 사용하여 코드를 간결하게 작성할 수 있다.
   ex : `str.replace('#', '1').replace('*', '0');`
3. 문자열 자르기 (Substring)
   substring(start, end): start 인덱스부터 end 전까지의 문자열을 반환한다.

\*추가 : 인덱스를 계산하는 대신, 매 반복마다 앞의 7자리를 잘라내고(`substring(0, 7)`), 원본 문자열을 남은 뒷부분으로 갱신(`s = s.substring(7)`)하는 방식을 사용할 수도 있다.
