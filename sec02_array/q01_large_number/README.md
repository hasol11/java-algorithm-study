# 1. 큰 수 출력하기

-> N개의 정수를 입력받아, 자신의 바로 앞 수보다 큰 수만 출력하는 프로그램 작성

### 1) 로직 흐름

첫 번째 수는 비교 대상이 없으므로 무조건 출력(추가)한다.

1번 인덱스부터 끝까지 반복하며, 현재 수(arr[i])가 바로 앞의 수(arr[i-1])보다 큰 경우에만 출력 목록에 추가한다.

---

### 2) 핵심 개념

- 인덱스 제어 (Start from 1)
  arr[i-1](앞의 수)과 비교해야 하므로, 반복문을 0이 아닌 1부터 시작해야 한다.
  0부터 시작할 경우 arr[-1]이 되어 IndexOutOfBoundsException 에러가 발생한다.

- ArrayList
  조건에 맞는 수만 골라내야 하므로, 결과의 개수가 유동적이다. 따라서 크기가 고정된 배열보다 ArrayList를 사용하는 것이 편리하다.
