# 4. 피보나치 수열

-> 1) 피보나치 수열을 출력한다. 피보나치 수열이란 앞의 2개의 수를 합하여 다음 숫자가 되는 수열이다.
-> 2) 입력은 피보나치 수열의 총 항의 수 N이 주어진다.

### 1) 로직 흐름

1. 배열 생성 및 초기화:

- N개의 항을 저장할 int 배열을 생성한다.
- 피보나치 수열의 첫 번째와 두 번째 항은 무조건 1이다. 따라서 arr[0] = 1, arr[1] = 1로 직접 값을 넣어준다.

2. 반복문 순회 (인덱스 2부터):

- 이미 0, 1번 인덱스는 채웠으므로, i = 2부터 n까지 반복한다.
- 점화식 적용: arr[i] = arr[i-2] + arr[i-1] (전전 항 + 전 항 = 현재 항) 공식을 사용하여 값을 채운다.

3. 출력: 완성된 배열을 순회하며 출력한다.
