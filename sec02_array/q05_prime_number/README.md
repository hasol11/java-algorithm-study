# 5. 소수 (에라토스테네스의 체)

-> 자연수 N이 입력되면 1부터 N까지의 소수의 개수를 출력하는 프로그램 작성

### 1) 로직 흐름

이 문제는 소수 판별을 위해 일일이 나누어보는 방법을 쓰면 시간 초과가 발생한다.
대신 "배수 지우기" 방식인 에라토스테네스의 체를 사용해야 한다.

1.  **배열 생성**:
    - `N+1` 크기의 체크용 배열 `ch`를 생성하고 0으로 초기화한다. (인덱스 번호가 곧 숫자)
2.  **순회 (2부터 N까지)**:
    - `i`가 2부터 N까지 반복한다.
    - 만약 `ch[i] == 0`이면?
      - **소수 발견되면**: `i`는 지워지지 않았으므로 소수다. -> `answer` 증가 (`++`)
      - **배수 지우기**: `i`의 배수들(`i`, `2i`, `3i`...)을 찾아 배열에서 체크(`ch[j] = 1`)한다.

---

### 2) 핵심 개념

1. 에라토스테네스의 체 (Sieve of Eratosthenes)

- 마치 체(Sieve)로 걸러내듯이, 소수를 찾으면 그 소수의 **배수들을 모두 지워나가는(체크하는)** 방식.
- 남은 숫자들은 나누어떨어지는 수가 없다는 뜻이므로 소수가 된다.
- 대량의 소수를 한꺼번에 판별할 때 가장 빠르고 효율적인 알고리즘이다.
