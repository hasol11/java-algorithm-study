# 6. 뒤집은 소수

-> N개의 자연수가 입력되면, 각 자연수를 뒤집은 후 그 뒤집은 수가 소수이면 그 소수를 출력하는 프로그램 작성

### 1) 로직 흐름

1.  **숫자 뒤집기 (Mathematical Reversing)**:
    - 입력받은 숫자(`tmp`)를 0이 될 때까지 반복하며 10으로 나눈 나머지(`%`)와 몫(`/`)을 이용해 뒤집는다.
    - `res = res * 10 + t` 공식을 사용하여 자릿수를 하나씩 올려준다.
2.  **소수 판별 (Prime Check)**:
    - 뒤집어진 수(`res`)가 소수인지 확인하는 별도의 메서드 `isPrime(int num)`을 호출한다.
    - **예외 처리**: `num`이 **1인 경우 false**를 반환한다. (1은 소수가 아님)
    - **약수 확인**: 2부터 `num-1`까지 반복하며 나누어떨어지는 수가 있으면 소수가 아니므로(`false`) 리턴한다.
3.  **결과 저장**:
    - 소수로 판별된 수만 `ArrayList`에 추가하여 반환한다.

---

### 2) 핵심 개념

#### ① 숫자를 뒤집는 알고리즘

문자열(`String`)로 변환하여 뒤집을 수도 있지만, **나머지 연산(`%`)**과 **나눗셈(`/`)**을 이용하는 수학적 방식이 알고리즘 문제해결의 정석이다.

```java
int res = 0;
while(tmp > 0) {
    int t = tmp % 10;      // 1. 일의 자리 추출
    res = res * 10 + t;    // 2. 기존 결과 자리 올림 후 더하기
    tmp = tmp / 10;        // 3. 추출한 일의 자리 제거
}
```
