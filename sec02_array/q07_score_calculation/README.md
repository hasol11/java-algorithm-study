# 7. 점수 계산

-> 연속으로 맞을 경우 가산점을 부여하여 점수를 계산하는 프로그램 작성

### 1) 로직 흐름

이 문제는 앞의 결과를 확인하기 위해 배열을 뒤적거릴 필요 없이, 가중치 변수(`cnt`) 하나만 관리하면 아주 쉽게 풀린다.

1.  변수 초기화:
    - `answer`: 총점을 저장할 변수
    - `cnt`: 현재 연속으로 맞은 횟수(가산점)를 저장할 변수 (초기값 0)
2.  순회 및 점수 계산:
    - 배열을 0번부터 끝까지 순회한다.
    - **정답(1)인 경우**: `cnt`를 1 증가시키고, 그 값을 `answer`에 누적한다.
    - **오답(0)인 경우**: 연속 정답이 끊겼으므로 `cnt`를 0으로 초기화(Reset)한다.

---

### 2) 핵심 개념

#### ① 가중치 누적

- 과거의 인덱스(`i-1`)를 참조하여 계산할 수도 있지만, 상태를 유지하는 변수 `cnt`를 사용하는 것이 훨씬 직관적이다.
- `cnt`는 "지금까지 연속으로 몇 번 맞았는지"를 기억하는 역할을 한다.

#### ② 상태 리셋

- 0(오답)을 만나는 순간 연속 기록은 깨지게 된다.
- 이때 `cnt`를 0으로 만들어주어야 다음 정답이 나왔을 때 다시 1점부터 시작할 수 있다.
