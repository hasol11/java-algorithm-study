# 7. 등수 구하기

-> N명의 학생의 국어점수가 입력되면 각 학생의 등수를 입력된 순서대로 출력하는 프로그램을 작성

### 1) 로직 흐름

1. 이중 for문 (점수 비교):
   바깥쪽 루프는 등수를 구할 대상 학생 i를 선택한다.
   대상 학생의 등수를 일단 1등으로 초기화하고
   내부 루프는 다른 모든 학생 j와 점수를 비교한다.
   만약 arr[j] > arr[i]라면, 나보다 점수가 높은 사람이 있다는 뜻이므로 rank를 1 증가시킨다.

2. 결과 출력:
   내부 루프가 끝나면 확정된 rank를 출력하고 다음 학생으로 넘어간다.

---

### 2) 핵심 개념

등수는 "나보다 점수가 높은 사람의 수 + 1"과 같다.
예를 들어, 나보다 잘 본 사람이 0명이면 나는 1등이고, 3명이면 나는 4등이 된다.
이 원리를 이용하면 같은 점수가 있을 때도 자동으로 공동 등수가 처리된다.
