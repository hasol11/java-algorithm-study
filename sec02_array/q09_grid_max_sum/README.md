# 9. 격자판 최대합

-> N이 입력되면, N\*N 격자판이 생성되고, 각 행의 합, 각 열의 합, 두 대각선의 합 중 가장 큰 합 출력하는 프로그램

### 1) 로직 흐름

1. 초기화: 최댓값을 저장할 변수 answer를 Integer.MIN_VALUE(가장 작은 정수값)로 초기화한다.

2. 행과 열의 합 구하기 (이중 for문):
   sum1 += arr[i][j]: i행이 고정된 상태에서 열(j)이 이동하므로 각 행의 합이 된다.
   sum2 += arr[j][i]: i열이 고정된 상태에서 행(j)이 이동하므로 각 열의 합이 된다.

3. 두 대각선의 합 구하기 (단일 for문):
   sum1과 sum2를 다시 0으로 초기화한다.
   루프(i)가 0부터 N-1까지 도는 동안 인덱스의 규칙성을 이용해 대각선의 합을 누적한다.

---

### 2) 오답 정리

- 이전 접근 :
  - 행, 열, 대각선 합을 구하기 위해 for문을 각각 따로 돌림.
  - 최댓값을 찾을 때 변수에 값을 담아 일일이 대조함.

- 개선된 접근 :
  - 루프 통합 : arr[i][j], arr[j][i]을 활용해, 하나의 이중 for문 안에서 행과 열의 합을 동시에 구함.
  - Math.max()를 사용해 코드를 간소화 함.
