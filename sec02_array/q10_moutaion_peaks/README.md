# 10. 봉우리

-> 높이가 적혀있는 N\*N 격자판에서 상하좌우 숫자보다 큰 숫자인 봉우리의 개수를 구하는 프로그램

### 1) 로직 흐름

[나의 풀이: 가장자리 패딩(Padding) 방식]

1. 가장자리를 0으로 감싸기 위해 (N+2) \* (N+2) 크기의 2차원 배열을 생성한다.

2. Java는 배열을 new로 생성할 때 자동으로 모든 요소를 0으로 초기화하므로, 별도의 처리 없이 가장자리가 0으로 세팅된다.

3. 인덱스 1부터 N까지 이중 for문을 돌며 안쪽 영역에만 실제 격자 데이터를 입력받는다.

4. 다시 인덱스 1부터 N까지 탐색하며, 현재 위치 arr[i][j]가 상하좌우(i-1, i+1, j-1, j+1)의 값보다 모두 엄격히 큰지 if문으로 확인하고, 조건을 만족하면 count를 증가시킨다.

장점: 경계값을 벗어나는지 확인하는 복잡한 조건식이 필요 없어 코드가 직관적이고 실행 속도가 빠르다.

[참고: 방향 배열(dx, dy)을 활용한 풀이]

1. 상하좌우 이동을 위한 방향 배열 dx = {-1, 0, 1, 0}, dy = {0, 1, 0, -1}을 선언한다.

2. 이중 for문 내부에서 k를 4번 반복하며 다음 탐색 좌표인 nx = i + dx[k], ny = j + dy[k]를 구한다.

3. nx와 ny가 배열의 유효한 인덱스 범위 (0 <= nx < N && 0 <= ny < N) 안에 있는지 경계선 체크를 한 후 대소 비교를 진행한다.

장점: 탐색 방향이 8방향 등으로 늘어나거나 복잡한 미로 찾기(DFS/BFS) 문제로 발전할 때 확장이 매우 용이한 알고리즘의 표준 패턴이다.

---

### 2) 오답 정리

- 발생 오류: ArrayStoreException: java.lang.Integer

- 원인: 2차원 배열(int[][])을 0으로 채우기 위해 Arrays.fill(arr, 0)을 직접 사용함. 자바의 2차원 배열은 '1차원 배열 객체'를 담는 배열이므로, 그 자리에 정수형 기본값(0)을 넣으려 하면 타입 불일치 오류가 발생한다.

- 배열 자동 초기화: Java에서 new int[N][N] 형태로 배열을 생성하면, 모든 요소가 자동으로 기본값인 0으로 초기화된다. 따라서 0으로 채우기 위한 별도의 코드는 작성할 필요가 없다.
- 특정 값으로 2차원 배열 초기화: 만약 0이 아닌 다른 값(예: -1)으로 초기화해야 한다면, 향상된 for문을 사용하여 각 행(1차원 배열)마다 Arrays.fill()을 적용해야 한다.

````java
// 0이 아닌 다른 값으로 2차원 배열을 초기화하는 올바른 방법
for (int[] row : arr) {
    Arrays.fill(row, -1);
}```
````
