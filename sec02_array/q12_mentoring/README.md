# 11. 멘토링

-> N명의 학생들이 M번의 수학테스트를 진행해 만들 수 있는 멘토와 멘티 짝의 수를 구하는 프로그램

### 1) 로직 흐름
1. 멘토와 멘티가 될 수 있는 확률은 n*n 이다. 따라서 이중 포문으로 경우의 수를 돌아야 한다.
2. 3번의 테스트를 진행해서 각각 등수를 구하고 멘토의 등수가 멘티의 등수보다 작은 경우에 cnt를 1 증가시킨다.
3. cnt가 m과 동일하면 모든 시험에서 충족한 것이므로 answer을 하나 증가시킨다.

---

### 2) 알게 된 내용

1. 완전탐색 문제이다: 입력값의 크기가 아주 작은 것을 봤을 때 모든 경우의 수를 다 따져봐도 시간이 충분하다.
