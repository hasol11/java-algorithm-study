# 2. 공통 원소 구하기

-> A,B 두 개의 집합이 주어졌을 때 공통 원소를 추출하여 오름차순으로 출력하는 프로그램

### 1) 로직 흐름

1. 먼저 입력값을 받아 각각 a, b 배열에 저장한다.
2. Arrays.sort()를 통해 a,b를 오름차순으로 정렬한 후 p1, p2 포인터를 정의한다. 각 포인터는 배열마다 존재하며 포인터가 옮겨가며 비교한다.
3. 만약에 a가 b보다 작다면 a 포인터를 1 증가시키고, 만약 a가 b보다 크다면 b 포인터를 1 증가시킨다. 그리고 만약 a랑 b가 같다면 answer에 추가하고 포인터 둘다 1씩 증가시킨다.
4. `p1 < n && p2 < m` 둘 중 하나라도 끝에 도달하면 멈춘다.

### 2) 핵심 개념

브루트포스: O(n×m)
투 포인터(정렬 포함): O(n log n + m log m)
