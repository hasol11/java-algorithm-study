# 4. 연속 부분수열

-> N개로 이루어진 수열에서 연속부분수열의 합이 특정 숫자 M이 되는 횟수를 구하는 프로그램

### 1) 로직 흐름

1. 부분수열의 시작(lt)과 끝(rt)을 가리킬 두 개의 포인터를 0으로 초기화한다.
2. rt를 오른쪽으로 한 칸씩 이동하며 해당 인덱스의 값을 부분수열의 합(sum)에 누적한다.
3. sum이 목표 숫자 M과 같아지면 정답(answer) 카운트를 1 증가시킨다.
4. sum이 M보다 크거나 같은 동안 반복문을 돈다.
5. lt가 가리키는 값을 sum에서 빼고, lt를 오른쪽으로 한 칸 이동(증가)시켜 윈도우 크기를 줄인다. 윈도우를 줄이는 과정에서도 sum이 M과 같아지면 answer 카운트를 1 증가시킨다.

### 2) 핵심 개념

for문을 사용하면 O(n^2)의 시간복잡도를 가지지만, lt와 rt 두 개의 포인터를 사용하면 O(n)의 시간복잡도로 문제 해결할 수 있다.
