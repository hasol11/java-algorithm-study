# 4. 연속된 자연수의 합

-> N개로 이루어진 수열에서 연속부분수열의 합이 특정 숫자 M이 되는 횟수를 구하는 프로그램

### 1) 로직 흐름

1. 연속된 자연수들의 합으로 특정 숫자를 만들 때, 수열을 구성하는 숫자는 목표값의 절반을 초과할 수 없다. 따라서 전체 탐색 범위를 (목표값 / 2) + 1로 제한하여 전체 연산량을 대폭 줄인다.
2. 오른쪽 포인터(rt)를 한 칸씩 전진시키며 배열의 값을 누적합(sum)에 더한다.
3. 누적합이 목표값과 같거나 커질 경우, 왼쪽 포인터(lt)가 가리키는 값을 누적합에서 차감하고 lt를 전진시키며 윈도우(구간)의 크기를 좁힌다.

### 2) 내 풀이와의 차이

초기 작성한 코드는 두 개의 포인터를 사용해 잘 구했다고 생각했으나, 첫번째 포인터를 고정시켜서 불필요한 반복이 일어났다. 또한 n/2+1 보다 더 큰 값은 아무리 해도 두 개 이상의 합으로 n을 만들 수 없다는 점을 간과했던 것 같다.
투 포인터와 슬라이딩 윈도우 개념을 잘 기억해놓자.
