# 6. 최대 길이 연속부분수열

-> 0, 1로 구성된 수열이 주어졌을 때, 0을 최대 k번까지 1로 바꿀 수 있다. 이때 만들 수 있는 1로만 이루어진 부분수열의 최대 길이를 구하는 문제

### 1) 로직 흐름

1. 투 포인터 `rt`(앞에서 이동)와 `lt`(뒤따라가는 포인터)를 사용한다.
2. 현재 구간의 길이는 `rt - lt + 1`이다.
3. `rt`를 한 칸씩 옮기면서, 만나는 값이 0이면 "1로 바꾼 것"으로 간주하고 `cnt`를 1 증가시킨다.
4. `cnt`가 k를 초과하면, `lt`를 오른쪽으로 옮기며 구간을 줄인다. 이때 `lt`가 가리키는 값이 0이면 `cnt`를 1 감소시키고 `lt`를 한 칸 전진시킨다.
5. 매 `rt` 이동마다 현재 구간 길이(`rt - lt + 1`)와 지금까지의 `answer`를 비교해 더 큰 값으로 갱신한다.
