# 1. 학급 회장 (해쉬)

-> 반 학생들이 투표를 진행한 후 어떤 기호의 후보가 학급 회장이 되었는지 출력하는 프로그램

### 1) 로직 흐름

1. key값은 Character, value는 Integer의 해쉬맵을 정의한다. 
2. 표 결과인 String 문자열을 문자 단위로 순회하면서 해쉬맵에 기록한다. `getOrDefault(키값, 0) + 1` 을 사용하여 특정 문자가 처음 나왔을 때는 0 + 1, 또 나왔을 때는 횟수 + 1로 계산되도록 한다.  
3. `map.keySet()`과 반복문을 사용해 해쉬맵에 등록된 모든 후보를 탐색한다. 각 key 값의 value를 max랑 비교해 제일 큰 key값을 구한다.
4. 구한 key값이 제일 많은 표를 받은 것임으로 학급 회장이다. 

### 2) 개념 

1. `HashMap<K,V>` : 데이터를 키와 값의 쌍으로 저장하는 자료구조 
2. `map.getOrDefault(x,0)` : 개수를 카운팅할 때 중요, x에 해당하는 값을 가져오는데 만약 값이 없다면 0을 반환
3. `map.containKey('A')` : A를 포함하고 있는지 True, False로 준다 
4. `map.size()` : 키의 개수를 알려준다 
5. `map.remove('A')` : 'A'라는 키로 저장된 값을 맵에서 삭제
6. `for(char key:map.keySet()){}` : 맵에 있는 모든 키들을 추출하여 반복문으로 탐색
