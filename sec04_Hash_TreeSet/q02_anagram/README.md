# 2. 아나그램

-> 길이가 같은 두 개의 단어가 주어지면 두 단어가 아나그램인지 판별하는 프로그램

### 1) 로직 흐름

1. s1을 한 문자씩 순회하며 HashMap에 `문자(key) : 등장 횟수(value)`로 저장한다. 
2. s2도 한 문자씩 순회하며 확인한다. s2의 문자 x가 map에 있으면 value를 1 감소시킨다.
3. 이때 다음 두 경우 중 하나라도 해당하면 "NO"를 리턴한다. 
- map에 아예 없는 문자인 경우
- value가 0인 경우 (s1보다 s2가 해당 문자가 더 많은 경우)
4. s2가 끝까지 순회하고도 걸리는 게 없다면 동일한 것이므로 "YES"를 리턴한다.

### 2) 개념
`map1.equals(map2)`: 동일하면 true 리턴
`map.containsKey(x)`: 키값이 x인 값이 map에 있는지
