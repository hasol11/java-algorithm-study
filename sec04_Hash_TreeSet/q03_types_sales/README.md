# 3. 매출액의 종류

-> 구간별 매출액의 종류를 출력하는 프로그램

### 1) 로직 흐름

1. 배열 첫 번째 요소부터 k-1번째 요소까지 미리 HashMap에 담는다.
2. rt를 k-1 위치부터 시작해 새로 들어온 요소를 맵에 추가한다.
3. 완성된 윈도우의 크기를 `map.size()`를 통해 확인하고 answer에 저장한다.
4. 윈도우를 다음칸으로 이동시키기 위해 현재 윈도우의 가장 왼쪽 요소를 빼준다.
5. 위의 내용을 반복한다.

### 2) 내 풀이와의 비교
HashMap을 매번 새로 정의하면서 개수를 세도록 설계했었는데, 사실 그렇게 하면 시간복잡도가 더 커진다. 
rt와 lt에서는 무조건 처음 값을 계산해놓고 그 이후에 한 칸씩 이동한다는 것을 생각하자.
