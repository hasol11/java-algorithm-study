# 4. 모든 아나그램 찾기

-> S문자열에서 T문자열과 아나그램이 되는 S의 부분문자열의 개수를 구하는 프로그램

### 1) 로직 흐름

1. 문자열 T의 알파벳 수를 해시맵에 기록한다.
2. S 문자열 탐색용 해시맵을 만들고, T의 길이보다 1개 적게 문자를 초기 세팅한다.
3. S 문자열을 순회하며 오른쪽 문자를 윈도우에 하나 추가한다.
4. 두 해시맵을 비교하여 일치하면 아나그램 개수를 1 증가시킨다.
5. 다음 탐색을 위해 윈도우 맨 왼쪽 문자의 빈도수를 1 감소시키고, 값이 0이 되면 해시맵에서 완전히 삭제한다.
6. 왼쪽 포인터를 이동시키며 계속 과정을 반복한다.
