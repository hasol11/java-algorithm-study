# 5. K번째 큰 수

-> N장의 카드 중 3장을 뽑아 만들 수 있는 모든 합계 중, 중복을 제외하고 K번째로 큰 수를 구하는 프로그램

### 1) 로직 흐름

1. 중복을 제거하고 데이터를 내림차순으로 자동 정렬해 줄 TreeSet을 생성한다.
2. for문을 사용하여 N장의 카드 중 3장을 뽑는 모든 경우의 수를 탐색한다.
3. 탐색하며 구한 3장의 카드 합을 TreeSet에 추가한다. 이때 이미 나온 합은 자동으로 걸러지고 큰 수부터 정렬된다.
4. 모든 경우의 수 탐색이 끝나면, TreeSet에 저장된 값들을 순회하며 순서를 카운트한다.
5. 카운트가 K와 일치할 때, 해당 위치의 값을 정답으로 기록하고 순회를 멈춘다.
6. 만약 TreeSet의 모든 값을 확인했는데도 K번째 값을 찾지 못했다면 정답을 -1로 기록한다.

### 2) 핵심 개념 
1. TreeSet
TreeSet 자체는 중복을 없애주고 정렬을 해주는 자료구조다. 
여기에 괄호 안 옵션으로 Collections.reverseOrder()를 넣어주면 큰 숫자부터 차례대로 줄 세우는 규칙이 적용된다. 그 후에는 .add()로 값을 추가하기만 하면 알아서 중복 제거와 정렬이 된다.
