# 1. 올바른 괄호

-> 괄호가 입력되면 올바른 괄호이면 "YES", 올바르지 않으면 "NO"를 출력하는  프로그램

### 1) 로직 흐름

1. 주어진 문자열을 앞에서부터 한 글자씩 순회한다.
2. 여는 괄호 (를 만나면 스택에 넣는다(push).
3. 닫는 괄호 )를 만나면 스택에서 최상단 요소를 뺀다(pop).
4. 단, 빼기 전에 스택이 비어있다면(닫는 괄호가 더 먼저/많이 나온 경우) 짝이 맞지 않는 것이므로 즉시 "NO"를 반환한다.
5. 모든 문자열 순회가 끝난 후 스택이 비어있는지 확인한다. 비어있다면 모든 짝이 맞은 것이므로 "YES" 반환하고, 괄호가 남아있다면 "NO" 반환한다.

### 2) 핵심 개념 

1. 스택 : 후입선출 자료구조 LIFO
