# 2. 괄호문자제거

-> 입력된 문자열에서 소괄호 () 사이에 존재하는 모든 문자를 제거하고 남은 문자만 출력하는  프로그램

### 1) 로직 흐름

1. 문자를 담을 Stack을 정의하고, 주어진 문자열을 탐색한다.
2. 현재 문자(x)가 닫는 괄호 )가 아니면 스택에 push 한다.
3. 만약 닫는 괄호 )라면, 짝이 맞는 여는 괄호 (가 나올 때까지 스택에서 요소를 pop 하여 제거한다.
4. pop() 메서드는 꺼낸 값을 반환하므로, while (stack.pop() != '('); 와 같이 조건식을 세우면 여는 괄호를 꺼낼 때까지 반복해서 문자를 지울 수 있다.
5. 탐색이 끝나면 스택에 남은 문자들을 인덱스 순서대로(get(i)) 가져와 하나의 문자열로 합친 후 반환한다.

### 2) 핵심 개념 

1. stack.size() : 스택의 원소 개수 
2. stack.get(i) : i 번째 스택에 들어있는 값 출력 
3. stack.push() : add로 사용했는데, 이건 List에서도 사용하는 메서드이니까 스택을 사용하고 있다면 push를 사용하자.
