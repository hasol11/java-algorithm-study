# 3. 크레인 인형 뽑기

-> 크레인을 작동하여 인형을 뽑고, 동일한 인형이 담기면 터뜨려 사라진 인형의 개수를 구하는 프로그램

### 1) 로직 흐름

1. 2차원 배열로 주어진 기계의 상태와 크레인의 작동 위치(moves)를 입력받고, 뽑은 인형을 담는 용도로 스택을 생성한다.
2. moves 배열을 순서대로 순회하며 크레인을 작동시킨다. 맨 위에서부터 아래로 탐색하다가 0이 아닌 값을 만나면 해당 인형을 tmp 변수에 담고, 기계에서 그 위치는 0으로 바꾼다.
3. 스택이 비어있지 않고 바구니 맨 위에 있는 인형이 방금 뽑은 tmp와 같다면 두 인형을 터뜨린다. 이때 스택의 맨 위 값을 pop 해서 빼내고 정답 변수를 2 증가시킨다. 반대로 인형 모양이 다르거나 스택이 비어있다면 tmp를 스택에 새롭게 넣는다. 인형을 하나 처리했으면 반복문을 빠져나와 다음 작동 위치로 넘어간다.
