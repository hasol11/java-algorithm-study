# 4. 후위식 연산

-> 후위연산식이 주어지면 연산한 결과를 출력하는 프로그램

### 1) 로직 흐름

1. 입력받은 문자열을 한 글자씩 순회하며 확인한다.
2. `Character.isDigit()`을 이용해 숫자인지 판별하고, 숫자라면 스택에 넣는다. 만약 문자가 연산자라면 스택에서 두 개의 값을 꺼내어 계산하고 그 결과를 다시 스택에 넣는다. 
3. 마지막으로 스택에 남아있는 값이 후위식 연산의 최종결과이다.

### 2) 알아두면 좋은 점

1. char형 숫자를 int로 변환할 때의 주의점
문자형(char) 숫자를 (int)로 직접 강제 형변환하면, 문자 본래의 숫자가 아닌 아스키(ASCII) 코드 값이 반환된다. 예를 들어 문자 '2'는 정수 50으로 변환된다.
따라서 실제 정수값으로 사용하려면 
(1) 변수 x에서 48을 빼거나 문자 0을 빼기 
(2) `Character.getNumericValue(x)` 메서드를 사용
