# 5. 쇠막대기

-> 여러 개의 쇠막대기를 레이저로 절단했을 때 쇠막대기의 개수를 구하는 프로그램

### 1) 로직 흐름

1. 입력받은 문자열을 인덱스(i)를 사용해 한 글자씩 순회하며 확인한다.
2. 만약에 문자가 '('라면 파이프가 시작되거나 레이저 시작 지점이므로 우선 stack에 넣는다.
3. 그게 아니라면 (문자가 ')'라면) 짝을 맞추기 위해 stack에서 한 개를 꺼낸다. 이때 바로 앞 문자가 '('라면 레이저이므로 현재 스택에 겹쳐 있는 파이프 개수(`stack.size()`)만큼 answer에 더한다. 만약 바로 앞 문자가 ')'라면 파이프 하나가 끝나는 지점이므로 자투리 1조각이 남게 되어 answer에 1을 더한다.
